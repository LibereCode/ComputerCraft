# ComputerCraft README

This is a collection of lua code for the mod [ComputerCraft: tweaked](https://tweaked.cc/).

Just clone it to your turtle/computers save-directory, which is at (with PrismLauncher):
`~/.local/share/PrismLauncher/instances/{YOUR_INSTANCE_NAME}/minecraft/saves/{YOUR_WORLD_NAME}/computercraft/computers/{COMPUTER_ID}/`;
or just replace `~/.local/share/PrismLauncher/instances/` with `~/path/to/your/minecraftData/`.
